# Vue PDF View 
```
使用vue cli构建的Vue项目，主要用来在线预览pdf文件

@vue/cli 4.5.9
node v12.16.1
npm 6.13.4
```

## Project setup 
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
